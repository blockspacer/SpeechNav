Speech Pedestrian Navigation System of PITT Campus - Interactive System Design
In the applicaiton, users can input source and destination address by Voice Commands. The Navigation System is designed for the main buildings in PITT Campus.
After you speak building names of source and destination, and then saying command "Get Path" can give you the routes.
Commands: SET DESTINATION/SET SOURCE/GET PATH/START ROUTE/END ROUTE
Debug Console
A
Speak a Building Name of PITT. Command: set source
B
Speak a Building Name of PITT. Command: set destination
